# vless-caddy

Конфиг для начинающих и инструкция по использованию Caddy как реверс-прокси.
Для примера все делаем вручную, чтобы что-то в голове осталось.

## Реверс-прокси (Caddy)

Шаг 1. Установим Docker:
```bash
sudo curl -fsSL https://get.docker.com | sh
```

Шаг 2. Создадим папку для Caddy и перейдем в нее:
```bash
mkdir -p /opt/caddy && cd /opt/caddy
```

Шаг 3. Открываем `Caddyfile` в репе и смотрим содержимое. Надо подставить домен на свой и сохранить:
```bash
nano Caddyfile
```

Шаг 4. Открываем `docker-compose.yml` и копируем содержимое. С таким же названием переносим в папку Caddy:
```bash
nano docker-compose.yml
```

Шаг 5. Запустить реверс-прокси:
```bash
docker compose up -d
```

## Конфиги Xray

1. Скопировать серверный конфиг в панель. Серверный конфиг вставляется целиком, без изменений.
2. В клиентском конфиге заменить IP и домен на свои, после чего вставить в клиент.

## Cloudflare (опционально)

Этот же конфиг можно использовать через прокси Cloudflare. Для этого:

1. Переведите у домена облачко в "оранжевое".
2. Смените IP вашего сервера на `8.47.69.111`
